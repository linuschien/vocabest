# 領域術語表 (Domain Glossary)

## 核心領域物件 (Core Domain Objects)

### 使用者與學習歷程 (User & Learning Progress)
| 術語 | 英文 / 代碼 | 定義 | 備註 / 約束 |
|------|------------|------|------------|
| **一般使用者/學習者** | `User` / `Learner` | 系統操作與學習的主體，僅能存取自身資料。 | |
| **系統管理員** | `Admin` | 擁有系統最高權限，可存取字庫、題庫及所有使用者資料。 | |
| **目標學制** | `TargetLevel` | 系統推播題庫的難度基準。 | Enum: `JUNIOR_HIGH` (國中會考 1200/2000字), `SENIOR_HIGH` (高中學測 7000字) |
| **今日進度** | `DailyProgress` | 當日完成的題數統計。 | |
| **連續學習天數** | `LearningStreak` | 連續登入並完成測驗的天數，提供微型成就感。 | 以火焰圖標表示。如中斷則歸零或依特定規則遞減。 |
| **錯題本** | `ErrorLog` | 紀錄使用者答錯的單字與相關題目。 | 每答錯一次，對應單字的錯誤權重 (+1)。 |

### 題庫與測驗 (Question Bank & Quizzes)
| 術語 | 英文 / 代碼 | 定義 | 備註 / 約束 |
|------|------------|------|------------|
| **字庫** | `WordBank` | 系統收錄的所有英文單字清單與屬性。 | |
| **靜態題庫** | `StaticQuestionBank` | 預先由 AI 生成的關聯式資料庫題庫。 | 確保前端毫秒級載入，避免即時生成的延遲。 |
| **測驗題目** | `QuizQuestion` | 測驗的基本單位，包含情境克漏字與四個選項。 | 必須包含: 英文情境句(挖空)、中文翻譯、1個正解、3個誘答項。 |
| **情境克漏字** | `ContextualCloze` | 帶有空格的英文句子，讓使用者透過上下文推敲單字。 | |
| **間隔重複演算法** | `SpacedRepetition` | 依據使用者的錯誤權重，排定單字在隔日或特定天數後的優先複習佇列。 | 確保單字能在遺忘曲線的關鍵點進行複習。 |
| **歷屆考頻** | `ExamFrequency` | 該單字在歷年大考（學測/會考）中出現的頻率。 | 通常以 1~5 星標示。高頻字需產生較多變化的題型。 |
| **詞類變化與衍生字** | `Inflection` / `Derivative` | 單字的時態（過去式/進行式）、單複數，或是同字根的不同詞性（如 success/succeed）。 | 測驗中須確保選項符合情境句的文法。 |

## 功能術語 (Feature Terms)
| 術語 | 英文 / 代碼 | 定義 |
|------|------------|------|
| **一鍵啟動** | `QuickStart` | 首頁上的快速入口，點擊即跳過設定進入測驗心流。 |
| **無限制刷題** | `UnlimitedPractice` | 測驗過程中無體力值或愛心限制。 |
| **AI 解析面板** | `AIExplanationPanel` | 答錯時彈出的解析，包含選項差異、字根字首與聯想記憶法。 |
| **錯題特訓模式** | `ErrorReviewMode` | 針對即將遺忘的單字進行精準打擊的特殊測驗模式。 |
