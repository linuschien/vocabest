# 領域術語表 (Domain Glossary)

## 核心領域物件 (Core Domain Objects)

### 使用者與學習歷程 (User & Learning Progress)
| 術語 | 英文 / 代碼 | 定義 | 備註 / 約束 |
|------|------------|------|------------|
| **一般使用者/學習者** | `User` / `Learner` | 系統操作與學習的主體，僅能存取自身資料。 | |
| **系統管理員** | `Admin` | 擁有系統最高權限，可存取字庫、題庫及所有使用者資料。 | |
| **目標學制** | `TargetLevel` | 系統派送題庫的難度基準。 | Enum: `JUNIOR_HIGH` (國中會考 1200/2000字), `SENIOR_HIGH` (高中學測 7000字) |
| **今日進度** | `DailyProgress` | 當日完成的題數統計。 | |
| **連續學習天數** | `LearningStreak` | 連續登入並完成測驗的天數，提供微型成就感。 | 以火焰圖標表示。如中斷則歸零或依特定規則遞減。 |
| **單字熟練度** | `WordMastery` | 紀錄使用者對特定單字的掌握狀態。 | 包含錯誤權重 (Error Weight) 與下次複習日期 (Next Review Date)。每個使用者對每個單字僅有一筆狀態。 |
| **錯題紀錄** | `ErrorEvent` | 紀錄使用者具體答錯了哪一道題目。 | Append-only 的歷史紀錄，關聯至具體的 `QuizQuestion`。 |

### 題庫與測驗 (Question Bank & Quizzes)
| 術語 | 英文 / 代碼 | 定義 | 備註 / 約束 |
|------|------------|------|------------|
| **字庫** | `WordBank` | 系統收錄的所有英文單字清單與屬性。 | 單字可具備多重詞性，使用一般字典慣用縮寫 (如: `noun`, `verb`, `adj.`, `adv.`)。 |
| **靜態題庫** | `StaticQuestionBank` | 預先由 AI 生成的關聯式資料庫題庫。 | 確保前端毫秒級載入，避免即時生成的延遲。 |
| **測驗題目** | `QuizQuestion` | 測驗的基本單位，包含情境克漏字與四個選項。 | 必須包含: 英文情境句(挖空)、中文翻譯、1個正解、3個誘答項。 |
| **情境克漏字** | `ContextualCloze` | 帶有空格的英文句子，讓使用者透過上下文推敲單字。 | |
| **間隔重複演算法** | `SpacedRepetition` | 依據使用者的錯誤權重，排定單字在隔日或特定天數後的優先複習佇列。 | 確保單字能在遺忘曲線的關鍵點進行複習。 |
| **歷屆考頻** | `ExamFrequency` | 該單字在歷年大考（學測/會考）中出現的頻率。 | 通常以 1~5 星標示。高頻字需產生較多變化的題型。 |
| **詞類變化與衍生字** | `Inflection` / `Derivative` | 單字的時態（過去式/進行式）、單複數，或是同字根的不同詞性（如 success/succeed）。 | 測驗中須確保選項符合情境句的文法。 |

## 功能術語 (Feature Terms)
| 術語 | 英文 / 代碼 | 定義 |
|------|------------|------|
| **一鍵啟動** | `QuickStart` | 首頁上的快速入口，點擊即跳過設定進入測驗心流。 |
| **無限制刷題** | `UnlimitedPractice` | 測驗過程中無體力值或愛心限制。 |
| **AI 解析面板** | `AIExplanationPanel` | 答錯時彈出的解析，包含選項差異、字根字首與聯想記憶法。 |
| **錯題特訓模式** | `ErrorReviewMode` | 針對即將遺忘的單字進行精準打擊的特殊測驗模式。 |
| **語音朗讀** | `TextToSpeech` / `TTS` | 透過喇叭圖示觸發的發音功能。 | 使用瀏覽器 Web Speech API 或靜態音檔。 |
| **關鍵任務日誌** | `CriticalMissionLog` | 獨立於測驗心流外的瀏覽介面，以單字卡形式呈現特定時間內的 `ErrorEvent` 歷史紀錄。 | 取代傳統「錯題本」的負面稱呼。 |
| **支線任務** | `SideQuest` | 輔助核心測驗的遊戲化學習模組，用於增強學習趣味性。 | 包含 Wordle, Crossword 等。 |
| **Wordle 拼字猜謎** | `Wordle` | 透過顏色回饋（綠、黃、灰）猜測單字正確拼寫的機制。 | 考驗精確拼寫能力。 |
| **Crossword 填字遊戲** | `Crossword` | 透過線索（如中文翻譯或情境句）與字母交錯提示來填寫單字的網格遊戲。 | 考驗字義連結與拼寫。 |
| **翻牌對對碰** | `MemoryGame` | 透過翻牌配對英文單字與中文翻譯的記憶遊戲。 | 強化單字與語義的快速連結。 |
| **重新洗牌** | `DynamicShuffle` | 連續配對失敗達門檻時，未配對卡牌自動隨機交換位置的機制。 | 增加地獄難度的挑戰性。 |
| **聽音辨位賓果** | `AudioBingo` | 透過聽取語音並在網格中選出正確單字的賓果遊戲。 | 強化聽力與音形連結。 |
| **保底提示** | `HintGlow` | 單字連續答錯達一定次數後，系統自動在正確方塊上顯示的視覺引導特效。 | 用於降低兒童學習挫折感。 |
| **三星評等** | `StarRating` | 基於錯誤次數與提示使用狀況給予的過關評價（1至3顆星）。 | 用於遊戲化激勵。 |
