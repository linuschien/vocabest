# US-04 個人化錯題追蹤 (Personalized Error Tracking)

## 背景 (Background)
透過演算法針對使用者的弱點進行記錄與排程，達到最高效率的單字記憶。

---

## US-04-01：錯題自動收錄與演算法排程

**身份**：系統 (System)

> **As a** 系統，  
> **I want to** 當使用者答錯題目時自動將該單字寫入「專屬錯題本」並增加錯誤權重，依據間隔重複演算法排定優先複習佇列，  
> **So that** 使用者能在遺忘曲線的關鍵點精準複習。

### 驗收條件 (Acceptance Criteria)
- **AC1**：使用者每一次答錯，系統背景必須非同步地更新資料庫，將該單字的錯誤權重數值 +1。
- **AC2**：系統應具備排程演算法 (Spaced Repetition)，依據錯誤權重與上次複習時間，計算該單字的下次複習日期 (Next Review Date)。
- **AC3**：當使用者進入「錯題複習專區」時，題目的順序必須由 Next Review Date 為今日(或已過期)且錯誤權重最高的單字優先出現。
- **AC4**：當使用者在錯題複習專區答對該錯題時，系統應適當降低該單字的錯誤權重，或延後其下一次複習日期。
