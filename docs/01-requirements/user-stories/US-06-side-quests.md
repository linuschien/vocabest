# US-06 支線任務與遊戲化 (Gamified Side Quests)

## 背景 (Background)
為了增加學習的趣味性與留存率，系統除了核心的情境克漏字測驗外，額外提供「支線任務 (Side Quests)」，包含 Wordle (拼字猜謎) 與 Crossword (填字遊戲)。這些小遊戲能幫助使用者從不同維度（精確拼寫、字義聯想）鞏固單字記憶，提供有別於一般測驗的心流體驗。

---

## US-06-01：Wordle 拼字猜謎

**身份**：使用者 (Learner)

> **As a** 使用者，  
> **I want to** 遊玩類似 Wordle 的拼字猜謎遊戲，  
> **So that** 我能透過嘗試與顏色回饋，挑戰並驗證自己對單字精確拼寫的記憶。

### 驗收條件 (Acceptance Criteria)
- **AC1 (遊戲初始化)**：系統從使用者的目標單字庫或錯題佇列中，隨機挑選特定長度（例如 5~7 個字母）的單字作為謎底，並提供對應長度的空白網格與虛擬鍵盤供使用者輸入。
- **AC2 (顏色回饋驗證)**：使用者送出猜測後，系統根據字母與位置給予標準的顏色回饋：綠色 (字母存在且位置正確)、黃色 (字母存在但位置錯誤)、灰色 (謎底不含此字母)。同時虛擬鍵盤上的按鍵也要同步更新顏色狀態。
- **AC3 (次數限制與提示)**：使用者預設有 6 次猜測機會。當剩餘次數較少時，系統可允許使用者點擊提示按鈕，顯示該單字的中文翻譯或詞性。
- **AC4 (遊戲結算)**：使用者猜中單字或次數用盡時遊戲結束。畫面須展示該單字的完整資訊（含中文翻譯、發音與解析），並可一鍵進入下一局。

## US-06-02：Crossword 填字遊戲

**身份**：使用者 (Learner)

> **As a** 使用者，  
> **I want to** 遊玩 Crossword 填字遊戲，  
> **So that** 我能透過字義提示與字母交錯的線索來回想單字，增強單字意義與拼寫的連結。

### 驗收條件 (Acceptance Criteria)
- **AC1 (謎題呈現)**：系統載入包含橫向 (Across) 與直向 (Down) 的網格拼圖。使用者介面需清楚呈現題目列表與對應的輸入框。
- **AC2 (線索提示)**：點擊特定網格或題目時，高亮顯示對應的單字輸入區塊，並展示線索（線索可以是中文翻譯、同義詞、或是情境克漏字）。
- **AC3 (互動與輸入)**：使用者可在選取的網格中輸入字母，焦點會自動移至下一格。若網格與其他單字交錯，輸入的字母會同時反映在兩個單字中。
- **AC4 (驗證與完成)**：使用者填完所有空格後，系統自動驗證（或提供「提交」按鈕）。若有錯誤，標示錯誤的單字或格子；全部正確時，觸發通關動畫，並顯示學習統計獎勵。

## US-06-03：Memory Game 翻牌對對碰

**身份**：使用者 (Learner)

> **As a** 使用者，  
> **I want to** 遊玩極簡版的翻牌對對碰小遊戲，  
> **So that** 我能在輕鬆的遊戲過程中，透過「英文單字 ↔ 中文翻譯」的語義配對，加深對單字的印象。

### 驗收條件 (Acceptance Criteria)
- **AC1 (極簡配對邏輯)**：遊戲採用「英文單字 ↔ 中文翻譯」的配對模式（例：「Apple」配對「蘋果」），不需額外依賴圖片或反義詞庫。
- **AC2 (難度級別與動態盤面)**：系統依據所選難度，從現有單字庫隨機抽取單字生成盤面。遊戲全程「無開局預覽」且「無時間限制 (不會 Game Over)」，難度差異設計如下：
  - **簡單 (Easy)**：4組單字 (8張牌)，配對錯誤時卡牌停留 1.5 秒後蓋回。
  - **中等 (Medium)**：6組單字 (12張牌)，配對錯誤時卡牌停留 1.0 秒後蓋回。
  - **困難 (Hard)**：8組單字 (16張牌)，配對錯誤時卡牌極速蓋回 (0.5秒)。
  - **地獄 (Hell)**：10組單字 (20張牌)，配對錯誤時極速蓋回 (0.5秒)；且每連續配對失敗 3 次，場上尚未配對的卡牌會觸發「重新洗牌 (Dynamic Shuffle)」隨機交換位置。
- **AC3 (純前端無狀態)**：此為休閒支線任務，遊戲進度與計數僅存在前端狀態 (Local State)，不寫入後端資料庫，不記錄成績，亦不影響既有學習進度。
- **AC4 (互動回饋與結算)**：玩家每次翻開兩張牌，配對成功時卡牌保持翻開並亮起成功提示（如綠色高亮），並透過瀏覽器原生的 Web Speech API 自動朗讀該英文單字；配對失敗則短暫延遲後自動蓋回。遊戲中需提供「翻牌次數」計數器與正向「計時器」。完成所有配對後，祝賀畫面需顯示該局的「耗時」與「總步數」，並提供「再玩一次」按鈕讓玩家自我挑戰。

## US-06-04：聽音辨位賓果 (Audio Bingo)

**身份**：使用者 (Learner)

> **As a** 使用者，  
> **I want to** 遊玩「聽音辨位賓果 (Audio Bingo)」遊戲，  
> **So that** 我能在低門檻且充滿正向回饋的遊戲化情境中，加強英文單字的聽力辨識能力與音形連結。

### 驗收條件 (Acceptance Criteria)
- **AC1 (動態盤面與難度選擇)**：系統提供兩種難度模式：
  - **低年級模式 (3x3)**：產生 9 宮格，適合單字量較少階段。
  - **高年級模式 (4x4 或 5x5)**：產生 16 宮格或 25 宮格，增加干擾項與挑戰性。
  開局時系統從專屬錯題本或指定單字庫隨機挑選單字填入，卡牌位置皆隨機打亂 (Shuffle)。
- **AC2 (語音播放與節奏)**：回合開始或前一題完成且停頓約 1~1.5 秒後，系統自動呼叫 Web Speech API 播放當前目標單字發音。畫面上提供顯眼的「喇叭圖示」，使用者可不限次數手動重聽發音。
- **AC3 (容錯與提示機制)**：
  - **答錯溫和回饋**：點錯方塊時觸發 CSS Shake 輕微搖晃動畫，方塊暫時反灰，不跳出「X」或扣分。同時自動再次朗讀正確答案發音。
  - **保底光暈 (Hint Glow)**：同單字連續點錯達特定門檻（低年級 2 次，高年級 3 次），正確方塊自動觸發微光或跳動特效，引導完成點擊。
- **AC4 (遊戲化激勵與結算)**：
  - **微互動 (Micro-interactions)**：每次點對單字時，播放正向音效（如叮或金幣聲），而非僅在最後通關時給予。
  - **連線勝利判定**：依設定可為「單線 Bingo (橫/直/對角)」或「全盤掃除 (Full Clear)」。
  - **三星評等機制**：通關後依表現給評：⭐️⭐️⭐️ 3 顆星 (零失誤)、⭐️⭐️ 2 顆星 (點錯 1~2 次但無保底提示)、⭐️ 1 顆星 (使用保底通關)。
  - **視覺與音效獎勵**：完成 Bingo 時觸發全螢幕彩帶動畫與歡呼，並附加語音鼓勵 (如 "Excellent!")。
